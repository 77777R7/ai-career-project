# KB Seed Content

This folder is the structured content source for the MVP.

The first seed supports the V1 primary demo loop:

`Canadian university explorer -> Data / Business / Product / Software / Policy path -> 4-week roadmap -> Canadian Housing and Cost of Living Dashboard or adjacent project -> Project Step 1-2 -> Growth Snapshot update`

Content layers:

- `career_paths/`: structured career path entities.
- `skills/`: skill definitions and evidence examples.
- `roadmaps/`: milestone and action templates by path and stage.
- `projects/`: project builder templates.
- `actions/`: reusable tasks for roadmaps and projects.
- `metrics/`: Growth Snapshot scoring rules.
- `sources/`: source registry and license metadata.
- `relations.yaml`: entity relationships.
- `taxonomies.yaml`: shared enums.

Use `CONTENT_SCHEMA_ZH.md` as the schema contract.
